### Flarum 正體中文語言包

- ### 需求
Flarum v0.1.0-beta.6

- ### 安裝
```
composer require csineneo/flarum-ext-traditional-chinese
```

- ### 示例
https://vivaldi.club 

- ### 問題
若有任何問題，請[在此](https://vivaldi.club/t/flarum)提出
